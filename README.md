## About

A container lock system leveraging the Java Paper Spigot API. Noteworthy features encompass container sign lock system and a trust system.

* On Linux or macOS: `./gradlew build`
* On Windows: `gradlew build`

## Minecraft Version Supported

[Paper](https://papermc.io/software/paper) `1.20.X`
